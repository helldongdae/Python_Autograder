Python Autograder V. Alpha

Autograder for Python programming language designed by Jiwoong Song

HOW TO USE
1. Put every single hw .py files in hws folder
2. Change the input txt file according to your desired input
3. Change the output txt file according to your desired output
4. Type in student numbers in stunum.txt file, program will only grade students with specified student numbers.
5. Result of the program will come out in "result.txt" file after the program ends

WARNINGS
THIS PROGRAM CAN NOT CATCH INFINiTE LOOP
